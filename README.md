## Como rodar a aplicação

### Levante serviço do banco de dados

`docker compose up -d`

### Verifique se o serviço foi criado corretamente

`docker compose ps`

### Instale as dependências

`yarn install`

### Popule o banco de dados

`yarn seed`

### Rode a aplicação

`yarn dev`

### Abra o storybook

`yarn storybook`
